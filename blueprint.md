# Futbol AI App Blueprint

This document outlines the project structure, design, and features of the Futbol AI application.

## Overview

Futbol AI is a comprehensive soccer management application that provides users with a rich and interactive experience. It allows users to view player profiles, track team performance, and get AI-powered insights into upcoming matches.

## Style and Design

The application follows a modern and visually appealing design, with a focus on usability and clarity. Key design elements include:

*   **Dark Theme:** The app uses a dark theme with a vibrant color palette to create a premium and immersive experience.
*   **Custom Fonts:** The app uses Google Fonts to create a unique and expressive typography.
*   **Interactive UI Components:** The app uses a variety of interactive UI components, such as charts, graphs, and animations, to enhance the user experience.
*   **Responsive Layout:** The app is designed to be responsive and adapt to different screen sizes, ensuring a consistent experience across all devices.

## Features

The application includes the following features:

*   **Player Profile Screen:**
    *   Displays detailed player information, including name, number, level, and positions.
    *   Features a radar chart to visualize player stats.
    *   Includes a "Momentos Destacados" section to highlight key player moments.
*   **Command Center Screen:**
    *   Provides a centralized dashboard for managing the team.
    *   Includes a "Live Standings" table to track team performance in the league.
    *   Features a "Squad Status" view to get a quick overview of the team's roster.
    *   Includes an "AI Prediction" card to get AI-powered insights into upcoming matches.
    *   Displays an "Upcoming Match" card to provide information about the next game.
*   **Login Screen:**
    *   Provides a secure and easy-to-use login screen for users to access the application.
    *   Features a visually appealing design with a background image and styled UI components.
*   **Authentication Flow:**
    *   Implements a robust authentication flow to handle user login and session management.
    *   Uses Supabase for authentication and data storage.
